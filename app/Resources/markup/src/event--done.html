Reactive Music Apps in Angular and RxJS
Хочу послушать
5

Angular 2 combined with RxJS, @ngrx/store, and Immutable.js is a fantastic platform for reactive web applications. If we also add hot loading and the Web Audio API to the mix, we have something truly exciting: A platform for reactive music creation.

This talk is about creating musical systems on the Angular platform. We’ll see how we can model a process that generates music right in the browser and allows the user to control it on the fly. It’s a really enjoyable way to learn reactive Angular application development.
